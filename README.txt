Project Rapid Destroy - это игра в жанре Tower Defence.
В ней нужно строить, уничтожать врагов и многое другое!

Игра была вдохновлена: игрой Mindustry разработаной на Java.

Советую проверять папку GameStory в папке GameFiles, там есть файл Game.txt в котором есть информация
о концовке и доп. параметры. Также можно поменять значение в файле doNotChange.dat это дает особую
концовку и меняет Gameplay.

Еще поменять Gameplay может HardMode и DLC. Они делают игру интереснее и сложнее!

Если вы новичок советую при запуске, в настройках включить обучение!

Примерный вес игры: 168-190 МБ

Требуется установить:
pip install pygame

Project Rapid Destroy is a tower defense game.
In it you need to build, destroy enemies and much more!

The game was inspired by: Mindustry game developed in Java.

I advise you to check the GameStory folder in the GameFiles folder, there is a Game.txt file that contains information
about the ending and options. You can also change the value in the doNotChange.dat file, this gives a special
ending and changes Gameplay.

Gameplay can also be changed by HardMode and DLC. They make the game more interesting and more difficult!

If you are a beginner, I advise you to turn on training(Tutorial) in the settings at startup!

Approximate game weight: 168-190 MB

Required to install:
pip install pygame
